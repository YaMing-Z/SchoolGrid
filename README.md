# SchoolGrid

一个轻量化的 Web 排课与调课工具，核心解决"学期基准排课"与"日常临时调课"两大场景。

## 🌟 项目核心理念

本项目采用 **"Master & Delta"（基准与变动）** 架构：

- **Master (基准表)**：学期开始前生成的全局唯一排课方案。
- **Delta (变动集)**：记录每日因教师缺勤、出差导致的局部课位调整，支持一键撤销与历史追溯。

---

## ✅ 已实现功能

### 数据层

- **Excel 数据导入**：支持导入教师、班级、教学任务数据，提供模板下载
- **本地持久化**：Zustand + LocalStorage，无需后端

### 规则引擎（全面集成）

- **SubjectRule**：周课时数、连堂设置、每日最大节数（dailyMax）、上午/下午偏好（timePreference）
- **SubjectTimeRule**：学科固定课位（fixed）、学科绝对禁排（must_not）—— 排课与调课均生效
- **GlobalTimeRule**：全校禁排时段，自动注入所有教师 avoidTimeSlots
- **TeacherTimeRule**：教师个人绝对禁排（must_not）合并注入 avoidTimeSlots
- **TeacherLoadRule**：教师最大周课时上限，代课时强制检查

### 排课算法

- **贪心排课**：连堂课优先、固定课位优先、课程均匀分布、学科禁排硬过滤
- **均匀分布优化**：同一学科避免集中排在同一天

### 调课引擎（三级优先级）

1. **P0（首选）**：同班同日互换 —— 含学科禁排验证
2. **P1（次选）**：同班跨日互换 —— 含学科禁排验证
3. **P2（保底）**：同科目教师代课 —— 含教师课时上限检查

### 课表可视化

- 按班级 / 教师视图展示
- 规则配置交互界面（学科时间约束网格）

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📁 项目结构

```plaintext
src/
├── algorithms/          # 核心算法
│   ├── scheduler/       # 排课算法（greedyScheduler）
│   └── adjustment/      # 调课建议算法（P0/P1/P2）
├── components/          # UI 组件
├── data/               # 常量与配置
├── parsers/            # Excel 解析器
├── services/           # 服务层（ruleAggregator）
├── stores/             # Zustand 状态管理
└── types/              # TypeScript 类型定义
```
