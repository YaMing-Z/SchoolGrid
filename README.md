# SchoolGrid

一个轻量化的 Web 排课与调课工具，核心解决“学期基准排课”与“日常临时调课”两大场景。

## 🌟 项目核心理念

本项目采用 **“Master & Delta”（基准与变动）** 架构：

- **Master (基准表)**：学期开始前生成的全局唯一排课方案。
- **Delta (变动集)**：记录每日因教师缺勤、出差导致的局部课位调整，支持一键撤销与历史追溯。

## 🚀 核心功能

- **关系型数据导入**：支持 Excel 模板导入。完美处理教师与班级的 1:N 映射关系（如数学老师带 2 个班，体育老师带 6 个班）。
- **类 Redmine 过滤器规则引擎**：用户可自定义“学科必须在下午”、“教师每日课时上限”等硬性或软性约束规则。
- **三级优先级智能调课**：
  1. **P0 (首选)**：同班同日对调。
  2. **P1 (次选)**：同班本周内跨日对调。
  3. **P2 (保底)**：跨老师同科目代课。
- **纯前端架构**：无需后端数据库，数据存储于 LocalStorage，支持导出 `.json` 或 `.xlsx` 分享使用。

## 🛠 技术栈

- **框架**: React / Vue 3 (Vite)
- **状态管理**: Zustand / Pinia
- **表格处理**: SheetJS (js-xlsx)
- **样式**: Tailwind CSS
- **部署**: GitHub Pages
