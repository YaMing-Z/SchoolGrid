# SchoolGrid

一个轻量化的 Web 排课与调课工具，核心解决”学期基准排课”与”日常临时调课”两大场景。

## 🌟 项目核心理念

本项目采用 **”Master & Delta”（基准与变动）** 架构：

- **Master (基准表)**：学期开始前生成的全局唯一排课方案。
- **Delta (变动集)**：记录每日因教师缺勤、出差导致的局部课位调整，支持一键撤销与历史追溯。

## ✅ 已实现功能

- **Excel 数据导入/导出**：支持导入教师、班级、教学计划数据，导出排课结果
- **贪心排课算法**：支持连堂课、固定时段、课程均匀分布
- **三级优先级智能调课**：
  1. **P0 (首选)**：同班同日其他教师课程互换
  2. **P1 (次选)**：同班本周内跨日互换
  3. **P2 (保底)**：同科目教师代课
- **课表可视化**：按班级/教师视图展示，支持交互式调整
- **本地持久化**：Zustand + LocalStorage，无需后端

## 🛠 技术栈

- **框架**: React 18 + TypeScript + Vite
- **状态管理**: Zustand
- **表格处理**: SheetJS (xlsx)
- **样式**: Tailwind CSS

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📁 项目结构

```
src/
├── algorithms/          # 核心算法
│   ├── scheduler/       # 排课算法
│   └── adjustment/      # 调课建议算法
├── components/          # UI 组件
├── data/               # 常量与配置
├── parsers/            # Excel 解析器
├── services/           # 服务层
├── stores/             # Zustand 状态管理
└── types/              # TypeScript 类型定义
```
